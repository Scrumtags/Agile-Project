## Get Started

1. Clone this repository

```
git clone https://github.com/jyleoxcino/ACIT-2911-Group-22.git
```

2. Download Python 3.11.X


3. Install Dependencies

```
pip install pyqt5 requests
```

4. Run Studybuddy

```
python studybuddy.py
```
